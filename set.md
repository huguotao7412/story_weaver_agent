# 《苍澜修仙界》完整背景设定集

> **核心世界观引言：**
> “光阴者，百代之过客；万物者，岁月之熔炉。”
> 苍澜界天道恒定，损有余而补不足。凡夺天地造化以求长生者，必遭天妒，故有寿元之限、三灾九难之劫。唯有不争不抢、隐于红尘者，方能在大道长河中留下只鳞片爪。

## 一、 世界基础法则

* **灵气设定：** 天地间充斥着“五行灵气”。灵气并非取之不尽，高阶灵脉被顶级宗门垄断，散修只能在灵气稀薄的“凡俗界”或“外海”苦苦挣扎。
* **长生铁律：** 无论功法多强，境界不突破，大限一到必定坐化。延寿丹药极为珍贵，且同一人服用多次会产生“抗药性”（天道限制），强行续命会引来极其恐怖的“化道之劫”。
* **因果业障：** 修仙者杀戮过重、强夺机缘，会沾染“业障”。业障深重者，突破时极易走火入魔，引来心魔劫。这逼迫高阶修士行事需顾忌因果，也为主角的“苟道”提供了合理的外部生存环境。

## 二、 修真境界与主角的“岁月体验”

传统修士修的是“与天争命”，主角修的是“把天熬死”。以下为苍澜界正统境界，以及主角在各个境界的独特状态：

### 1. 炼气期（共九层）

* **常人：** 寿至百岁。引气入体，掌握基础法术，为了几块下品灵石在坊市里与人拼命，属于修仙界的最底层炮灰。
* **主角：** 停滞期长达**五百年**。主角因为《万古长青诀》锁住生机，容貌永远停留在十八岁。他混迹于凡间酒肆、铁匠铺，身上带着凡人时期买的“粗陶泥葫芦”**和**“生锈铁剑”。这五百年，泥葫芦装过酒、装过水，渐渐开始自动汇聚微薄的灵气，变成了能蕴养丹药的“灵葫”。

### 2. 筑基期（初期、中期、后期、大圆满）

* **常人：** 寿至两百五十载。需服用“筑基丹”度过雷劫，神识初生，可御剑飞行。是普通宗门的中坚力量。
* **主角：** 停滞期长达**三千年**。当昔日同门早化作黄土，主角终于慢吞吞地自动筑基。这三千年中，他常穿的一件“凡人青麻衣”，因为陪伴他吸收了三千年的日月精华，纤维发生了异变，化作了无视下品法器攻击的“岁月青衣”。他开始在各种修仙界大战的废墟中“捡垃圾”，种下各种难以成活的上古灵药种子。

### 3. 金丹期（丹分九品）

* **常人：** 寿至五百载。结成金丹客，方是我辈人。可辟谷，断肢重生，能够开辟自己的洞府，成为一方老祖。
* **主角：** 停滞期长达**两万年**。主角结出的是毫无灵力波动的“灰白石丹”。此时他的“岁月通灵”迎来了第一次质变：他获得了微弱的“光阴领域”。当年那把生锈铁剑，经过两万年无时不刻的灵力冲刷与岁月包浆，剑身上天然生长出“时光道纹”。主角出剑极慢，但凡被剑光扫中，敌人会在瞬间流失百年寿元。

### 4. 元婴期

* **常人：** 寿至两千载。破丹成婴，元婴不灭则肉身可夺舍重修。能施展瞬移，掌控部分天地元气。
* **主角：** 停滞期**十万年起步**。主角已经成了苍澜界真正的“活化石”。他院子里当年随手种下的一株“杂草”，已经长成了遮天蔽日的“十万年建木神树”。他不再需要动手，遇到元婴期的仇敌，只需扔出那个盘了十几万年的“粗陶泥葫芦”，葫芦里喷出的岁月长河之水，能瞬间把敌人的元婴冲刷成凡人。

### 5. 化神期及之上（炼虚、合体、大乘、渡劫）

* **常人：** 万载寿元，追求飞升灵界/仙界，是苍澜界真正的幕后棋手。
* **主角：** 主角已经跨越了纪元。整个苍澜界的宗门兴衰更迭了数十次。当化神老怪们为了飞升抢夺最后一件仙器打得天崩地裂时，主角正坐在院子里，拿那件所谓的“无上仙器”当铲子给药田松土——因为这仙器，不过是主角三十万年前随手丢弃的一把破柴刀，被岁月赋予了灵性后自行飞走的。

## 三、 主角专属核心设定：【万古长青诀】与【岁月通灵】

* **《万古长青诀》：**
* **来历：** 并非上古大能遗留，而是天地初开时，一株凡草感悟天地所化生的一块无字石碑。
* **特性一：绝对锁寿。** 一经修炼，寿元上限即被打破。体内灵气不再用于爆发，而是化作一潭死水，锁住所有生命精气。
* **特性二：龟速修炼与伪装。** 吸收灵气的速度是常人的百分之一。且外表看起来永远是气息孱弱的低阶修士，极具欺骗性。
* **代价：** 毫无攻击法门。在拥有“包浆神装”之前，主角的战斗力弱于同阶任何人。


* **天赋：【岁月通灵】（核心金手指）**
* **机制：** 一切常人眼中没有价值的“死物”（凡铁、普通木头、石头），只要被主角长年累月地贴身携带、把玩、注入那带着长生气机的《万古长青诀》灵力，就会在漫长的时光中打破品阶壁垒。
* **质变公式：** 凡物 + 主角温养 + 1000年 = 极品灵器；+ 10000年 = 通天灵宝；+ 100000年 = 蕴含先天大道的仙器。
* **限制：** 主角不能直接温养原本就强大的法宝。如果拿着别人炼制好的高级飞剑，温养无效。必须是从“凡物”或最低级的灵物开始，由主角亲自“盘”出来。这注定了主角只能走稳健苟道。

## 四、 苍澜界势力与地理格局

为了配合主角漫长的寿命，世界版图必须宏大，且存在兴衰更替。

* **中州神域（核心区）：** 灵脉最浓郁之地。被“一寺、二阁、四大圣地”把持。这里的人高高在上，垄断飞升资源。主角在前期（十万年内）绝不踏足此地，只在边缘观望。
* **四荒十二州（修仙界基本盘）：** 凡人国度与修仙宗门混杂之地，宗门林立，杀伐不断。主角常年隐居于此，以各种伪装身份（凡俗客栈老板、小修真家族的客卿、落魄散修）混迹其中，每隔几百年换一个身份和地方。
* **无尽渊海（妖族与禁地）：** 海洋深处藏有大恐怖。
* **葬仙古地（沧海桑田的见证）：** 苍澜界最神秘的禁地，传闻有真仙陨落。但对主角来说，所谓的禁地，其实只是五万年前他住过的一个山洞，里面布置的几个基础防御阵法，因为吸收了五万年的灵气，变异成了连化神老怪都破不开的“绝世杀阵”。

## 五、 长生流的经典看点与基调

1. **红尘炼心，岁月史书：** 主角会结交一些凡人或资质普通的修士朋友。看着他们意气风发，看着他们英雄迟暮，看着他们化作一抔黄土。主角在他们的坟前倒一杯浊酒，收走他们留下的遗物（凡品），将其作为新的“通灵”载体。
2. **跨越时光的复仇与报恩：**
* *仇敌：* 某宗门天骄抢了主角一块灵石并打伤他。主角微微一笑，转身离开。八千年后，该天骄所在宗门遭遇大劫，主角在万里之外，用盘了八千年的弓箭，射出了跨越虚空的一箭，直接将该宗门的山门钉死。
* *恩情：* 曾有一名女修在主角落魄时赠予他一枚低阶疗伤丹。一万年后，该女修的后代家族面临灭顶之灾，主角化作神秘大能降临，随手抛出一件仙器镇压全场，留下一句“故人因果，今日结清”，飘然而去。


3. **不争即是争：** 秘境开启，天下大乱。主角不进秘境，而是在秘境外开个小茶摊卖茶。等过了几万年，秘境彻底废弃，主角慢悠悠走进去，把当年大能们打碎的法宝残片、漏掉的边角料全部捡走，回家慢慢“盘”。
